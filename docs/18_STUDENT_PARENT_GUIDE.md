# 학생 및 보호자 안내

## 학생 안내

### 로그인 후 처음 할 일

1. `/login`에서 로그인합니다.
2. 필요하면 `/onboarding`에서 표시 이름, 학습 연령대, 비밀번호를 확인합니다.
3. `/student/dashboard`로 이동합니다.
4. 화면의 `Manual` 메뉴에서 HTML 매뉴얼을 확인할 수 있습니다.

### 학습 세션 참여

1. 오늘 공개된 세션을 선택합니다.
2. 글을 읽습니다.
3. 종이에 맥락 스케치를 작성합니다.
4. 제출 화면에서 스케치 이미지를 업로드합니다.
5. 자기 설명, 중요한 연결, 어려웠던 점을 입력합니다.
6. 제출합니다.

### 피드백 확인

- 튜터가 피드백을 공개하면 학생 화면에서 확인할 수 있습니다.
- `/student/portfolio`에서 공개된 피드백이 누적됩니다.
- 피드백은 점수만 보는 화면이 아니라 다음 스케치를 더 잘하기 위한 안내입니다.

## 보호자 안내

### 접속 조건

보호자가 학생 기록을 보려면 관리자가 보호자-학생 연결을 `승인` 상태로 저장해야 합니다.

### 대시보드

- URL: `/parent/dashboard`
- 연결된 학생 목록을 확인합니다.
- 학생별 공개 피드백을 확인합니다.
- 최근 피드백과 누적 성장 기록을 확인합니다.
- 보호자 요약 리포트를 인쇄하거나 PDF로 저장할 수 있습니다.
- 화면의 `Manual` 메뉴에서 HTML 매뉴얼을 확인할 수 있습니다.

### 보호자가 볼 수 있는 정보

- 학생 이름
- 공개된 세션 제출 피드백
- 튜터가 보호자에게 공개한 요약 내용
- 포트폴리오 누적 기록

### 보호자가 볼 수 없는 정보

- 관리자 사용자 관리 화면
- 튜터 내부 메모
- 다른 학생의 제출물
- Supabase 또는 Vercel 운영 설정

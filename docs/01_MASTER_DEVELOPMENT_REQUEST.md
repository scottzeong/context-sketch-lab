# Context Sketch Lab / SketchFlow v1 통합 개발 요청서

## 1. 제품 정의

`Context Sketch Lab`은 튜터가 학습자의 독해 사고를 설계하고, 학습자가 종이에 비언어적 맥락스케치를 수행한 뒤, 튜터가 그 결과를 관찰·해석·피드백하는 튜터링 기반 사고 훈련 플랫폼이다.

시스템명은 `SketchFlow`이며, AI 엔진명은 `Context Architecture Engine`이다.

v1에서 AI는 아이가 그린 그림을 직접 분석하지 않는다. 스케치의 의미 판단은 튜터가 입력한다. AI는 텍스트 생성, 텍스트 구조 분석, 튜터 피드백 기반 평가 및 리포트 초안 생성에 집중한다.

## 2. 핵심 가치

```text
- 튜터가 수업의 설계자이자 최종 판단자다.
- AI는 자동 채점자가 아니라 튜터의 문서화, 구조화, 피드백 작성을 돕는 보조 엔진이다.
- 학습자는 종이에 직접 그리고 연결하며 사고한다.
- 시스템은 학습자의 결과보다 사고 과정과 튜터의 관찰을 축적한다.
- 포트폴리오는 AI 추정치가 아니라 튜터 승인 결과를 기반으로 한다.
```

## 3. 주요 사용자

```text
admin: 조직, 사용자, 템플릿, 매뉴얼, AI 로그, 릴리즈 관리
tutor: 글 생성, 글 분석, 세션 설계, 학생 제출 확인, 관찰 입력, 평가/피드백 승인
student: 세션 확인, 글 읽기, 종이 스케치 작성, 사진 업로드, 간단한 설명 입력, 피드백 확인, 성찰 작성
parent: 승인된 포트폴리오와 성장 리포트 확인
```

## 4. 핵심 학습 흐름

### 튜터 수업 설계

```text
1. 튜터 로그인
2. 학습 그룹 선택
3. 주제, 연령, 난이도, 분량, 글 구조 선택
4. AI 학습용 글 생성
5. 튜터가 글 수정 또는 승인
6. AI 글 구조 분석 실행
7. 학습 목표와 활동지 템플릿 선택
8. 세션 저장 및 학생에게 배포
```

### 학생 활동

```text
1. 학생 로그인
2. 오늘의 세션 확인
3. 글과 활동 지시 확인
4. 종이 활동지에 비언어적 맥락스케치 작성
5. 스케치 사진 업로드
6. 짧은 자기 설명 입력
7. 튜터 검토 대기
```

### 튜터 검토

```text
1. 제출물 확인
2. 이미지와 학생 설명 확인
3. 튜터가 관찰 메모, 핵심 연결, 오해 가능성, 피드백 방향 입력
4. AI가 튜터 입력 기반 평가 초안 생성
5. AI가 학습자용 피드백 초안 생성
6. 튜터가 수정·승인
7. 학생에게 공개
8. 포트폴리오와 리포트 데이터로 저장
```

## 5. AI 기능 범위

v1에서 구현할 AI 기능은 다음으로 제한한다.

```text
- 학습용 글 생성
- 글 구조 분석
- 세션 질문/활동 지시문 초안 생성
- 튜터 관찰 입력 기반 루브릭 평가 초안 생성
- 학습자 피드백 초안 생성
- 누적 성장 리포트 초안 생성
```

제외:

```text
- AI 이미지 분석
- OCR 중심 손글씨 판독
- AI 자동 최종 채점
- 멀티 LLM provider
```

## 6. 기술 스택

```text
Frontend: Next.js App Router, TypeScript, Tailwind CSS, shadcn/ui, lucide-react, React Hook Form, Zod
Backend: Supabase Postgres, Supabase Auth, Supabase Storage, RLS
AI: OpenAI API only, server-side only, Structured Outputs
Deploy: Vercel
Repository: GitHub
```

## 7. 디자인 방향

디자인은 단순한 CRUD 관리자 화면이 아니라 `교육 연구실 + 튜터 워크벤치` 느낌을 목표로 한다.

```text
- 차분하고 전문적인 대시보드
- 튜터가 반복적으로 쓰기 편한 고밀도 작업 화면
- 글 생성, 분석, 피드백, 승인 흐름이 한눈에 이어지는 레이아웃
- 학생/보호자 화면은 간결하고 따뜻하게
- 리포트와 포트폴리오는 공유 가능한 문서 수준의 완성도
```

## 8. 완료 기준

```text
- 튜터가 AI로 글을 생성하고 수정할 수 있다.
- AI가 글의 구조를 분석해 세션 설계에 활용할 수 있다.
- 튜터가 학습 세션을 생성하고 배포할 수 있다.
- 학생이 세션을 확인하고 스케치 사진과 설명을 제출할 수 있다.
- 튜터가 스케치 관찰과 피드백을 입력할 수 있다.
- AI가 튜터 입력 기반 평가/피드백 초안을 생성한다.
- 튜터 승인 전에는 학생에게 최종 피드백이 공개되지 않는다.
- 결과가 학생 포트폴리오와 리포트 데이터로 누적된다.
- 매뉴얼과 릴리즈 노트가 함께 제공된다.
```

# User Manual 및 Release Notes 가이드

## 1. 매뉴얼 목표

User Manual은 단순 도움말이 아니라 실제 수업 운영 문서다.

대상:

```text
- 관리자
- 튜터
- 학생
- 보호자
```

## 2. 튜터 매뉴얼 목차

```text
1. Context Sketch Lab 개요
2. 학습 그룹 만들기
3. AI로 학습용 글 생성하기
4. 글 구조 분석 읽기
5. 학습 세션 만들기
6. 종이 활동지 사용하기
7. 학생 제출물 확인하기
8. 스케치 관찰 입력하기
9. AI 평가/피드백 초안 사용하기
10. 피드백 승인 및 공개하기
11. 포트폴리오와 리포트 확인하기
```

## 3. 학생 매뉴얼 목차

```text
1. 오늘의 세션 확인하기
2. 글 읽기
3. 종이에 맥락스케치 그리기
4. 사진 업로드하기
5. 자기 설명 입력하기
6. 튜터 피드백 확인하기
7. 성찰 작성하기
```

## 4. 보호자 매뉴얼 목차

```text
1. 자녀 포트폴리오 보기
2. 성장 리포트 읽기
3. 피드백 공개 상태 이해하기
4. 개인정보와 이미지 보관 정책 확인하기
```

## 5. Release Notes 원칙

릴리즈마다 다음 항목을 기록한다.

```text
- 새 기능
- 개선된 기능
- 수정된 문제
- AI 프롬프트 또는 schema 변경
- DB migration 변경
- 사용자에게 필요한 행동
```

## 6. v1 첫 릴리즈 기준

```text
- 튜터 글 생성/분석
- 세션 생성
- 학생 제출
- 튜터 리뷰
- AI 평가/피드백 초안
- 포트폴리오 저장
- 기본 리포트
- 매뉴얼 제공
```

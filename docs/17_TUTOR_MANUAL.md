# 튜터 매뉴얼

## 전체 흐름

```text
글 생성/분석
→ Text 저장
→ 그룹 관리
→ 세션 생성/공개
→ 학생 제출 확인
→ 튜터 리뷰
→ AI 피드백 초안 생성
→ 튜터 수정 및 공개
→ 리포트 작성
```

## AI Workbench

1. `/tutor/workbench`로 이동합니다.
2. 주제, 분량, 글 구조, 학습 목표를 입력합니다.
3. AI 글 생성을 실행합니다.
4. 생성된 글을 확인하고 필요하면 수정합니다.
5. 구조 분석을 실행합니다.
6. 결과가 수업에 적합하면 Text 저장소에 저장합니다.

## Text 저장소

- URL: `/tutor/texts`
- 제목, 본문, 목표, 구조, 난이도로 검색합니다.
- 상태, 구조, 난이도 필터로 자료를 좁힙니다.
- 저장된 글과 구조 분석 JSON을 확인합니다.

## 그룹 관리

- URL: `/tutor/groups`
- 수업반 또는 학습 그룹을 생성합니다.
- 학생을 그룹에 배정합니다.
- 세션은 그룹 단위 또는 전체 학생 대상으로 배포할 수 있습니다.

## 세션 생성

1. `/tutor/sessions/new`로 이동합니다.
2. 사용할 글을 검색하고 선택합니다.
3. 세션 제목, 그룹, 학습 목표, 활동지 템플릿을 확인합니다.
4. 상태를 `초안` 또는 `공개`로 선택합니다.
5. 저장합니다.

## 세션 관리

- URL: `/tutor/sessions`
- 제목, 글, 그룹, 목표로 검색합니다.
- 그룹/상태 필터를 사용합니다.
- 상태를 `초안`, `공개`, `종료`로 변경합니다.
- `복제`를 눌러 같은 수업 구성을 새 초안 세션으로 재사용합니다.

## 제출물 리뷰

1. `/tutor/submissions`에서 제출물을 선택합니다.
2. 학생 스케치 이미지를 확인합니다.
3. 학생 자기 설명과 연결 설명을 읽습니다.
4. 튜터 관찰, 강점, 오개념, 다음 과제를 입력합니다.
5. AI 피드백/평가 초안을 생성합니다.
6. 학생용 피드백과 보호자 요약을 수정합니다.
7. 최종 확인 후 공개합니다.

## 리포트

- URL: `/tutor/reports`
- 학생, 그룹, 기간으로 필터링합니다.
- 자동 생성된 리포트 초안을 편집합니다.
- `초안 저장`으로 Supabase에 저장합니다.
- `인쇄/PDF`로 출력합니다.

## 튜터 운영 원칙

- AI 결과는 초안입니다.
- 학생 그림 해석은 튜터가 책임집니다.
- 피드백 공개 전에는 학생과 보호자가 볼 수 없습니다.
- 반복 수업은 세션 복제로 시작하면 운영 시간이 줄어듭니다.
